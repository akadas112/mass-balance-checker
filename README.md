# Gemini API Mass Balance Checker

This version uses Gemini API instead of OpenAI, so it can work with Gemini's free API tier if available for your account/region.

Upload these files directly to the GitHub repository root:

- index.html
- server.js
- package.json
- README.md
- .env.example

Render settings:

Build Command: npm install
Start Command: npm start

Environment Variables on Render:

GEMINI_API_KEY = your Gemini API key
GEMINI_MODEL = gemini-2.5-flash

Do not upload your real API key to GitHub.
