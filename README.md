# Gemini API Mass Balance Checker with Fallback Models

This version uses Gemini API and tries fallback models if one model is temporarily unavailable.

Upload these files directly to your GitHub repository root:

- index.html
- server.js
- package.json
- README.md
- .env.example

Render settings:

Build Command: npm install
Start Command: npm start

Environment Variables on Render:

GEMINI_API_KEY = your Gemini API key
GEMINI_MODEL = gemini-2.5-flash-lite

Do not upload your real API key to GitHub.
